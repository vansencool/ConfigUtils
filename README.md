# ConfigUtils
Library which allows developers to easily manage yaml configurations

# How to install it?

**Gradle**

```groovy
repositories {
    maven { url 'https://jitpack.io' }
}
```

```groovy
dependencies {
    implementation 'com.github.vansencool:ConfigUtils:1.0.1'
}
```

**Maven**

```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```

```xml
<dependency>
    <groupId>com.github.vansencool</groupId>
    <artifactId>ConfigUtils</artifactId>
    <version>1.0.1</version>
</dependency>
```
